Intruksi Kerja Pembuatan FixOrder

Preparation

1. Tarik LPB dari HSIS dengan ketentuan :

  • Tanggal H-30 sd H-1 (tanggal awal bebas sd H-1, tergantung kebutuhan data (1 minggu, 2 minggu dsb)
  • Jenis item = semua
  • Urutan = kode barang
  • qty(0) = ya
  • data dibuka sebagai excel data only

jika fixorder ini dibuat tanggal 13 feb 2022 maka data yg ditarik adalah tanggal awal data yang dipilih sd 12 Feb

2. Tarik data sales target dengan ketentuan :

  • Tanggal sales = tanggal awal LPB pada poin1 di atas sd H-1
  • data dibuka sebagai excel data only

Menyusun Fix Order

1. Buka sheet "LPB"

  • Copy all data LPB yang sudah ditarik
  • Paste-kan ke sheet LPB di file fixorder

2. Buka Sheet "SALES"

  • Copy all data sales target yang sudah ditarik
  • Paste-kan ke sheet SALES di file fixorder

3. Buka Sheet "SETUP"

  • Isi tanggal awal periode fixorder yang akan dibuat di kotak D4
  • isi D15 sd D28 dengan rupiah proyeksi sales dengan ketentuan :
  • isi hanya hari H sampai dengan D28
  • jika fixorder ini dibuat di hari minggu maka proyeksi sales yang dimasukkan adalah dari hari minggu hingga jumat minggu berikutnya
  • Isi DATA STORAGE - isi dalam satuan M³ (konversi cm³ ke m³ adalah berapa cm³ dibagi satu juta)
  • Isi TABEL ORDER CYCLE
  • pilih Special Order atau Normal order pada kotak H14
  • "SPECIAL ORDER" jika order hari rabu lusa masuk ke periode sekarang
  • "NORMAL ORDER" jika order hari rabu lusa masuk ke periode order minggu selanjutnya
  • Kotak hijau hari rabu i25 jika ada orderan untuk minggu ini atau ikut periode minggu depannya lagi berarti diisi dengan angka "1"
  • Isi kotak I18 sd I32 dengan angka 1 untuk datang barang frozen chiller dan 0 untuk tidak datang
  • Isi kotak K18 sd K32 dengan angka 1 untuk datang barang dry dan 0 untuk tidak datang
Penting: Pastikan dua hal di atas dilakukan agar orderan tidak bergerak dikemudian hari atau setelah data LPB diupdate
